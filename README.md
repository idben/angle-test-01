# 角度學習系統

一個互動式的網頁應用程式，幫助學習者練習角度的概念與測量。

## 功能特色

- **互動式量角器**：透過 SVG 繪製的半圓量角器，提供視覺化的角度學習體驗
- **拖曳操作**：可拖曳量角器上的控制點來調整角度
- **角度吸附**：自動吸附到最近的 10 度刻度，方便精確定位
- **目標挑戰**：設定目標角度，挑戰自己是否能正確拖曳到指定角度
- **即時回饋**：顯示目前角度，並在檢查答案時提供成功/失敗的音效回饋
- **觸控支援**：同時支援滑鼠與觸控操作，適用於各種裝置

## 使用方式

1. 開啟 `index.html`
2. 在彈出的對話框中輸入目標角度（0～180 度）
3. 系統會隨機固定一個控制點，另一個控制點可以拖曳
4. 拖曳藍色控制點到正確的位置，使角度符合目標
5. 點擊「檢查答案」按鈕確認結果
