# 角度學習系統
## 目標
* 學習特定角度

## 預期技術
* HTML
* CSS
* JS
* 不使用 JS 框架
* 使用 2 個空白
* HTML/JS/CSS 檔案分離
* 使用 querySelector 系列語法，不使用 getElement 系列語法

## 預期操作
* HTML 中出現一個半圓
* 半圓上有刻度，每十度為一個單位
* 半圓週上出現兩個圓圈，分別在半圓的兩端
* 圓圈可以在半圓上拖曳，每十度為一個單位
* 兩個圓圈需要與半圓圓心造成一個角度，拖曳時需更新這個角度的顯示
* 有一個輸入欄位可以輸入度數，0～180
* 有一個按鈕按了可以檢查答案
* 檢查答案可以檢查兩個圓圈造成的角度是否與輸入欄位的數字一樣
